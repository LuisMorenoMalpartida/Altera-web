# altera (PHP + AJAX) para XAMPP

## Requisitos
- XAMPP instalado
- Apache encendido (Panel de Control de XAMPP)

## Instalación
1. Copia esta carpeta a: `C:\xampp\htdocs\altera`
2. Inicia Apache en XAMPP
3. Abre en el navegador: `http://localhost/altera/`

## Estructura
- `index.php`: layout principal
- `partials/`: header/hero/catalog/footer
- `api/items.php`: endpoint JSON para AJAX
- `js/`: JS modular (paginación separada)
- `css/`: estilos
- `legacy/`: respaldo del HTML de Wix
